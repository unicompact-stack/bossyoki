# BossYoki

## Что это
Проактивный тайм-менеджер — VK-бот с AI, который напоминает о задачах и помогает доводить дела до конца.

## Как запустить
1. Перейти в папку `code/`
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать .env файл с ключами
4. Запустить: `python main.py`

## Структура
- `idea/` — замысел и видение
- `architecture/` — чертёж проекта
- `business/` — «второй мозг» (постоянная база)
- `code/` — исходный код
