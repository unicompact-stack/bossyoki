# CLAUDE.md — BossYoki

## Описание проекта
BossYoki — проактивный тайм-менеджер. VK-бот с AI, который напоминает о задачах и помогает доводить дела до конца.

## Стек технологий
- **VK API** — для общения с пользователем
- **GitHub Models** — для AI-функций
- **Python** — основной язык
- **APScheduler** — планировщик напоминаний
- **requests** — HTTP-запросы

## Структура папок
- `idea/` — замысел проекта (11 файлов)
- `architecture/` — чертёж проекта (4 файла)
- `business/` — «второй мозг» (5 файлов)
- `code/` — исходный код (9 файлов)

## Роутинг
- Вопрос про «задачи / функции» → `code/`
- Вопрос про «архитектуру / структуру» → `architecture/`
- Вопрос про «бизнес / аналитику» → `business/`
- Вопрос про «идею / видение» → `idea/`

## Ключевые ограничения
1. **GitHub Models** — всегда использовать через GitHub Token
2. **.env** — хранить локально, не коммитить
3. **Коммит** — делать перед закрытием сессии
4. **Безопасность** — проверять (security review)
