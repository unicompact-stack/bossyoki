# BossYoki Code

## Запуск
1. Установить зависимости: `pip install -r requirements.txt`
2. Создать .env файл с ключами VK_TOKEN и GITHUB_TOKEN
3. Запустить: `python main.py`

## Структура
- `main.py` — точка входа
- `agent.py` — AI-агент
- `tools.py` — инструменты для задач
- `timer.py` — планировщик
- `vk_handler.py` — VK интеграция
- `prompts.py` — системный промпт
- `config.py` — загрузка конфигурации
